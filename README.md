# Data-Science-Lab-Assignment-Topsis

## TOPSIS Python Package

A Python implementation of the **TOPSIS (Technique for Order Preference by Similarity to Ideal Solution)** method for multi-criteria decision making.

This package ranks alternatives based on their distance from the ideal and negative-ideal solutions using user-defined weights and impacts.

---

## Features

- Command-line-based TOPSIS implementation
- Accepts CSV input files
- Supports custom weights and impacts
- Generates a ranked output CSV file
- Handles invalid inputs with meaningful error messages

---

## Installation

Install the package from PyPI using:

```bash
pip install Topsis-102317186-Kashish
```
